# Flight Prep and First Flights

This was the phase where all the abstract planning became real.

Before the first launch, I was excited but definitely uneasy. I had never flown this type of aircraft and I knew that one wrong assumption could end the flight before I even had time to react.

I was trying to be disciplined about checks: CG with the actual battery, control direction by hand, motor/prop direction, mode behavior at the aircraft, and arming logic. None of that is glamorous, but skipping any one of those is exactly how first launches go sideways.

My plan for the first attempt was conservative: use assisted mode, keep it calm, keep it simple, and do not push the envelope.

Even with that mindset, the takeoff ended in a crash. That was the moment where I stopped thinking in terms of "almost ready" and started treating validation like a hard requirement.
